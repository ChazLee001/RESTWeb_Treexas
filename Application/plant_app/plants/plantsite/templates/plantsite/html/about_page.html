About Us

Owl Team

This site contains information about the plants, ecoregions and stateparks of Texas. The purpose of this site is to provide reliable information about any and all plants that can be found in Texas. This site is meant for anyone looking to learn more about Texas plants and where to find them. This could include researchers, students, survivalists or foragers.

Datasources







Texas Plant Source

This datasource was scraped using the HTMLParser and BeautifulSoup Python packages. Information about each plant, such as botanical name, growing conditions, ecoregions and stateparks, were extracted from each plant

Texas State Park and Ecoregion Datasource

This datasource was scraped using the HTMLParser and BeautifulSoup Python packages. Latitude and longitude information was scraped for the state parks. Descriptions and native plants were scraped for each of the texan ecoregions


Global Biodiversity Information Facility

This datasource was used to obtain latitude and longitude information for plants using Selenium in python.


Texas Plant Information Database

This datasource was scraped using Selenium in python to obtain more plant information. It was also used to give each statepark and plant a ecoregion if it applied also using Selenium.

Tools


We used the Django web framework, along with Python to build our app.
We used HTML, CSS and some very simple Javascript to build templates for our web pages.
We used Google Cloud Platform to deploy app, and acquired domain name from namecheap.com.
We used MySQL as our database, and stored information on Cloud SQL, from Google Cloud Platform.
Data scraping was performed using the beautifulsoup Python package, while RESTful APIs were used when they were available.
We used Selenium for UI testing, Mocha for JavaScript testing and Django test for python testing.
We used Django to build our API, then used Postman to test it.
